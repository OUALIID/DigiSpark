<h2>3. Technical Specifications and Capabilities of Digispark</h2>
<h6>Digispark is a small development board that measures only 1.8 cm x 2.5 cm. It is powered by a 5V USB connection, and it has 6 I/O pins, including 3 analog pins. It can be programmed using the Arduino IDE, and it can be used to control LEDs, motors, and sensors.</h6>

<h2>4. Advantages and Disadvantages of Using Digispark</h2>
